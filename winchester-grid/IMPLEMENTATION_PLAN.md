# Visualisation Module Implementation Plan

## Goal
Create a web-based visualisation for grid allocation results using MapLibre GL JS and Turf.js.

## Architecture
- **Data Preprocessing (Python)**: Converts input GPKG and CSV files into web-friendly GeoJSON/JSON formats.
- **Frontend (HTML/JS)**: Loads the processed data and renders the map.

## Steps

### 1. Data Preprocessing [COMPLETED]
- Create `scripts/prepare_data.py`.
- Inputs:
    - `outputs/20251119_090902/deployment_allocations_assignment_trace.csv` (Allocations)
    - `data/8.3_solar_potential.gpkg` (Solar Parcels)
    - `data/8.4_wind_potential.gpkg` (Wind Parcels)
    - `data/primary_substation.gpkg` (Primary Substations)
    - `data/bsp_substations.gpkg` (BSP Substations)
    - `data/gsp_with_headroom.gpkg` (GSP Substations - if needed)
- Outputs (to `visualisation_module/data/`):
    - `parcels.geojson`: FeatureCollection of parcel polygons (with parcel_id).
    - `substations.geojson`: FeatureCollection of substation points (with substation_id/name).
    - `allocations.json`: Array of allocation objects (parcel_id, substation_id, year, status, etc.).

### 2. Frontend Development [COMPLETED]
- Create `index.html`:
    - Map container.
    - Legend/Controls (Updated to reflect Suitability and Substation types).
- Create `style.css`:
    - Basic styling.
- Create `app.js`:
    - Initialize MapLibre map (Using OpenStreetMap raster tiles).
    - Load GeoJSON/JSON data.
    - Use Turf.js to calculate centroids of parcels.
    - Generate connection lines (LineString) from parcel centroid to assigned substation.
    - **Styling**:
        - **Lines**:
            - Allocated: Thick, solid green line.
            - *Note: Non-allocated lines removed to improve performance and visibility.*
            - Opacity: Based on `deployment_year` (more opaque = sooner).
        - **Parcels**:
            - Fill color based on `Overall Suitability` (Green=TRUE, Yellow=MAYBE, Red=FALSE).
        - **Substations**:
            - Circle markers sized and colored by type (Primary=Red, BSP=Blue, GSP=Purple).

### 3. Implementation Details [COMPLETED]
- **Libraries**:
    - MapLibre GL JS (CDN)
    - Turf.js (CDN)
- **Color Scale**:
    - **Years**: Opacity interpolation (2025: 1.0 -> 2050: 0.2).
    - **Suitability**: Categorical (TRUE/MAYBE/FALSE).
    - **Substations**: Categorical (Primary/BSP/GSP).

### 4. Bug Fixes [COMPLETED]
- **Line Visibility Issue Fixed**:
    - Fixed substation coordinate extraction: Now handles both Point geometries and Polygon/MultiPolygon geometries by calculating centroids.
    - Improved year handling: Ensures deployment_year is converted to number and filters out null values.
    - Enhanced line styling: Added zoom-based line width scaling for better visibility at different zoom levels.
    - Layer ordering: Lines are now inserted before substations layer to ensure proper visibility.
    - Added debugging: Console logging to track line generation and identify missing data.
