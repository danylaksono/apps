apps

This repository hosts small interactive demos and playgrounds. The older Observable Framework project is now archived under /archive/observablehq.

City Rose

A Vite + React demo with a “city rose” visualization (deployed from /city-rose).

Walk (Isochrone)

A lightweight, pure JS isochrone generator that runs entirely in the browser (deployed from /walk).

Sector 7

An HTML+JS canvas game experience (deployed from /sector7).

Joyplot

An interactive deck.gl joyplot map visualization (deployed from /joyplot).

Archived ObservableHQ site

Legacy Observable Framework build artifacts and pages (kept for reference).

To edit these projects, check out the repository and run each demo from its own folder.
